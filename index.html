Instructions

Some common functions:
  • For addition, write +
  • For subtraction, write -
  • For multiplication, write *
  • For division, write /
  • For π, write Math.PI
  • For x2, write Math.pow(x,2)
  • For ln(x), write Math.log(x)
  • For logy(x), write Math.log(x)/Math.log(y)
  • For sin(x), write Math.sin(x)
  • For cos(x), write Math.cos(x)
